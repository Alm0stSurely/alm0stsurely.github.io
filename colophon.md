## Design principles

The type scale and spacing follow the fluid methodology described by [Utopia](https://utopia.fyi/) â€” no breakpoints, no arbitrary jumps. Sizes flow continuously between viewport widths using CSS `clamp()`, producing a scale that is mathematically coherent at any screen size.

Line lengths are held to approximately 66 characters (the "ideal measure" per Robert Bringhurst's *The Elements of Typographic Style*). Body text uses a line height of 1.5, headings tighten to 1.25, display text to 1.1 â€” following the general rule that as type gets larger, it needs proportionally less leading.

The color palette is deliberately restrained: warm darks, muted text, and a single accent derived from terracotta. The goal is to stay out of the way of the writing.
